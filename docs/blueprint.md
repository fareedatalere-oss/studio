# **App Name**: I-Pay

## Core Features:

- Prompt Input: Allow users to input a text prompt describing their desired application.
- AI-Powered Feature Generation: Use AI to generate a list of potential application features based on the user's prompt. The LLM uses a tool for filtering features to fit an MVP and Next.js architecture.
- Feature Selection and Editing: Allow users to select and edit the generated features, customizing the application to their specific needs.
- Feature Description: Provide clear descriptions for each generated feature to help users understand their functionality.
- JSON Export: Export the selected features as a JSON file, suitable for further development or use in other tools.

## Style Guidelines:

- Dark theme for the entire app.
- Welcome page to show 'Welcome to I-pay online business and transactions' then show sign up and sign in button.
- Primary color: Strong purple (#9D4EDD) to convey creativity and forward-thinking design.
- Background color: Soft lavender (#E9D5FF), a muted version of the primary color that is easy on the eyes.
- Accent color: Vibrant magenta (#E01A97) for highlighting key actions and interactive elements.
- Headline font: 'Space Grotesk', a geometric sans-serif, to give a tech-focused, computerized feel.
- Body font: 'Inter' for longer text, used to pair with Space Grotesk. A grotesque-style sans-serif with a modern, machined, objective, neutral look.
- Use clear, minimalist icons to represent features and actions within the application.
- A clean and structured layout with a focus on usability and easy navigation, suitable for desktop and mobile interfaces.
- Subtle transitions and animations to guide users through the application's workflow, such as loading animations during AI processing or smooth transitions between steps.
